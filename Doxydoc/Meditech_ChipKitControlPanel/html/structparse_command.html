char parseCommand::cmdData[160]

Unparsed Command String.

cmdData is the string after syntax checking. The string can contain one or more commands. When the command string is sent to the parser this value contains the command sequence. When the this string is empy all the commands has been processed and the system is ready to receive a new command.

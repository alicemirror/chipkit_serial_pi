typedef struct LCDTemplateField field

LCD template field type definition.

Defines a generic LCD display field

typedef word probeActive

Defines the active probe bit.

The two-bytes word has the corresponding probe bit when the probe is shown on the LCD display.
The bit order is the following:

  • 0x0001 - STETHOSCOPE_ON
  • 0x0002 - BLOODPRESS_ON
  • 0x0004 - HEARTBEAT_ON
  • 0x0008 - TEMPERATURE_ON
  • 0x0010 - ECG_ON
  • 0x0020 - Unused
  • 0x0040 - Unused
  • 0x0080 - Unused
  • 0x0100 - Unused
  • 0x0200 - Unused
  • 0x0400 - Unused
  • 0x0800 - Unused
